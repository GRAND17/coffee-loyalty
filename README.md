# ☕ Coffee Loyalty

Простое консольное приложение для учёта скидок постоянных клиентов кофейни.

## 🚀 Возможности

- Регистрация новых клиентов по номеру телефона
- Отметка визитов (заказ кофе)
- Автоматический расчёт скидки:
  - 0% при 0-4 визитах
  - 5% при 5-9 визитах
  - 10% при 10+ визитах
- Просмотр информации о клиенте
- Список всех клиентов с сортировкой по визитам

## 📋 Команды приложения

| Команда | Описание | Пример |
|---------|----------|--------|
| `reg Имя Телефон` | Регистрация нового клиента | `reg Анна +79161234567` |
| `check Телефон` | Отметить визит | `check +79161234567` |
| `info Телефон` | Информация о клиенте | `info +79161234567` |
| `list` | Показать всех клиентов | `list` |
| `clear` | Очистить экран | `clear` |
| `exit` | Выйти из программы | `exit` |
