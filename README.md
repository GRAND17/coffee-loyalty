\# ☕ Coffee Loyalty



Система учёта скидок для кофейни. Работает в командной строке.



\## 🚀 Что умеет



\- Регистрировать новых гостей

\- Считать визиты

\- Автоматически повышать скидку:

&#x20; - 0-4 визита = скидка 0%

&#x20; - 5-9 визитов = скидка 5%

&#x20; - 10+ визитов = скидка 10%

\- Показывать список всех гостей



\## 📋 Команды



| Команда | Что делает | Пример |

|---------|------------|--------|

| `reg Имя Телефон` | Регистрация | `reg Анна +79161234567` |

| `check Телефон` | Отметить визит | `check +79161234567` |

| `info Телефон` | Информация о госте | `info +79161234567` |

| `list` | Все гости | `list` |

| `exit` | Выход | `exit` |
